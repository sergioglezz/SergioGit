> 沒 Este tipo de operadores nos permiten comparar dos expresiones, devolvieﾌ］donos un booleano indicando si esa comparacioﾌ］ es verdadera o falsa.

Podemos encontrar los siguientes operadores de comparacioﾌ］:

* _Igual_ `$a == $b` - devuelve `TRUE` si $a es igual a `$b`

```php
$x = 100;  
$y = "100";
var_dump($x == $y); // returns true because values are equal
```
沒 **Resultado:**
```php
bool(true)
```

* _Ideﾌ］tico_ `$a === $b` - devuelve `TRUE` si $a es igual a `$b` y ademaﾌ《 son del mismo tipo.

```php
$x = 100;  
$y = "100";
var_dump($x === $y); // returns false because types are not equal
```
沒 **Resultado:**
```php
bool(false)
```

* _Diferente_ `$a != $b` - `TRUE` si `$a` no es igual a `$b`.

```php
$x = 100;  
$y = "100";
var_dump($x != $y); // returns false because values are equal
```
沒 **Resultado:**
```php
bool(false)
```

* _Diferente_ `$a <> $b` - `TRUE` si `$a` no es igual a `$b`.

```php
$x = 100;  
$y = "100";
var_dump($x <> $y); // returns false because values are equal
```
沒 **Resultado:**
```php
bool(false)
```

* _No Ideﾌ］tico_ `$a!==$b` - `TRUE` si `$a` no es igual que `$b`,o si no son del mismo tipo.

```php
$x = 100;  
$y = "100";
var_dump($x !== $y); // returns true because types are not equal
```
沒 **Resultado:**
```php
bool(true)
```

* _Menor que_ `$a < $b` - Es verdadero si `$a` es estrictamente menos que `$b`.

```php
$x = 10;
$y = 50;
var_dump($x < $y); // returns true because $x is less than $y
```
沒 **Resultado:**
```php
bool(true)
```

* _Mayor que_ `$a > $b` - `TRUE` si `$a` es estrictamente mayor que `$b`.

```php
$x = 100;
$y = 50;
var_dump($x > $y); // returns true because $x is greater than $y
```
沒 **Resultado:**
```php
bool(true)
```

* _Menor o igual que_ `$a <= $b` - `TRUE` si `$a` es menor o igual que `$b`.

```php
$x = 50;
$y = 50;
var_dump($x <= $y); // returns true because $x is less than or equal to $y
```
沒 **Resultado:**
```php
bool(true)
```

* _Mayor o igual que_ `$a >= $b` - `TRUE` si `$a` es mayor o igual que `$b`.

```php
$x = 50;
$y = 50;
var_dump($x >= $y); // returns true because $x is greater than or equal to $y
```
沒 **Resultado:**
```php
bool(true)
```

* _Nave espacial_ `$a <=> $b` - Un `integer` menor que, igual a, o mayor que cero cuando `$a` es respectivamente menor que, igual a, o mayor que `$b`. 汨_Disponible a partir de PHP 7_汨

```php
$x = 5;  
$y = 10;
echo ($x <=> $y); // returns -1 because $x is less than $y
echo "<br>";
$x = 10;  
$y = 10;
echo ($x <=> $y); // returns 0 because values are equal
echo "<br>";
$x = 15;  
$y = 10;
echo ($x <=> $y); // returns +1 because $x is greater than $y
```
沒 **Resultado:**
```php
-1
0
1
```

* _fusioﾌ］ de null_ `$a ?? $b` - El primer operando de izquierda a derecha que exista y no sea `NULL`. `NULL` si no hay valores definidos y no son `NULL`. 汨_Disponible a partir de PHP 7_汨

```php
//estas tres expresiones son equivalentes
$var = isset($_GET['foo']) ? $_GET['foo'] : "valor_por_defecto";
$var = $_GET['foo'] ?: "valor_por_defecto";
$var = $_GET['foo'] ?? "valor_por_defecto";
```

> 汨 _Para realizar la comparacioﾌ］ entre expresiones de diferente tipo, lo que se hace es igualar sus tipos._ Por ejemplo, si vamos a comparar un `string` que representa un nuﾌ［ero con un `integer`, los dos seraﾌ］ tratados como `integer`. El uﾌ］ico caso en el que no se realiza este cambio de tipos es cuando trabajamos con los operadores `===` y `!==`, ya que su utilizacioﾌ］ implica la comparacioﾌ］ de tipos tambieﾌ］.
